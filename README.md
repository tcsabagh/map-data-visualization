# Térképes adatvizualizáció

Ez az analisztika.hu blogon megjelent "Térképes adatvizualizáció Pythonban" poszt példa forráskódját tartalmazza.<br />

Ez az alkalmazás képes egy excel fájl GPS és egyéb adataiból egy HTML térképet generálni. A térkép mark pontjai reprezentálják azokat a pontokat, ahol a gépjármű áthaladt. Ehhez a pandas és folium Python könyvtárakat használtuk fel.
